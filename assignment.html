C Language Assignment with Sample Answers

1. C Language kya hai?
C ek general-purpose programming language hai jo Dennis Ritchie ne 1972 mein develop ki thi. Yeh fast, powerful, aur system programming ke liye bahut useful hai. Operating systems, embedded systems, aur applications banane ke liye use hoti hai.
2. C Language ke Features kya hain?
- Simple aur flexible language - Portable (alag-alag machines par chal sakti hai) - Fast execution - Low-level access (memory management ke liye) - Rich set of operators
3. C Program ka Basic Structure kya hota hai?
#include <stdio.h>

int main() {
    // Program ka code yahan likhen
    return 0;
}
4. Hello World Program likho:
#include <stdio.h>

int main() {
    printf("Hello, World!\\n");
    return 0;
}
5. Variables aur Data Types:
Variables memory locations hain jisme data store hota hai. Common data types: - int — integers ke liye - float — decimal numbers ke liye - char — single characters ke liye
6. Operators in C:
Arithmetic operators numbers par calculations karte hain. Examples: - + (addition) - - (subtraction) - * (multiplication) - / (division)
7. Comments kaise likhte hain C mein?
- Single line comment: // Yeh ek comment hai - Multi-line comment: 
/* 
Yeh multi-line 
comment hai 
*/
8. C Language ka Use Kahan Hoti Hai?
- Operating systems (jaise Linux, Windows ka kernel) - Embedded systems (microcontrollers, devices) - Game development - Compilers aur interpreters
Bonus Question: C Language ki History:
C language Dennis Ritchie ne Bell Labs mein 1972 mein develop ki thi. Yeh Unix operating system ko banane ke liye design ki gayi thi.
